- **[Advanced_Deep_Learning_4_Physics](https://github.com/scaioo/Advanced_Deep_Learning_4_Physics)** & **[Molecular_Dynamics_Simulations](https://github.com/scaioo/Molecular_Dynamics_Simulations)**: Advanced computational projects and simulations developed during my TUM courses.
